    APIs for Predict Products

    Make predictions for a product

    Request Example

    class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!"); 
    }
}

    Response Example

    {
    "code": 0,
    "msg": "",
    "data": {
      "transactionTime": 1660702505000,
      "transferAmountEv": 1000000000,
      "status": "Ok"
    }
}

    HTTP Request

    GET /phemex-predict/order/buy

    Request Parameters

    Parameter Required Type Comment
    productId true number product id
    currency true string currency
    amountEv true number amount ev value
    optionId true number option id

    Response Parameters

    Parameter Type Comment
    code number Code
    msg string Message
    data Object Data

    Errors

    The Phemex API uses the following HTTP codes and error codes:

    HTTP Code Meaning
    200 Request is valid
    403 Access denied
    404 Request path not found

    -62010

    Coupon does not exist

    -62011

    Coupon and product does not match

    -62012

    Conditions for using coupon not met
