    预测产品

    预测产品API

    预测产品申购

    请求示例

    class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!"); 
    }
}

    响应示例

    {
    "code": 0,
    "msg": "",
    "data": {
      "transactionTime": 1660702505000,
      "transferAmountEv": 1000000000,
      "status": "Ok"
    }
}

    HTTP 请求

    GET /phemex-predict/order/buy

    请求参数

    参数 必填 类型 备注
    productId true number 预测产品id
    currency true string 币种
    amountEv true number 转入金额
    optionId true number 预测选项id

    响应参数

    参数 类型 备注
    code number 返回码
    msg string 返回信息
    data Object 数据

    错误码

    Phemex使用以下HTTP Code和錯誤碼：

    HTTP Code 含義
    200 請求有效
    403 拒絕訪問
    404 訪問路徑不存在

    -62010

    加息券不存在

    -62011

    加息券的产品不匹配

    -62012

    加息券未到最低使用门槛
